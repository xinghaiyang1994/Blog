# Git flow

http://www.imooc.com/learn/751

![](https://ww1.sinaimg.cn/mw690/c13b829ely1fjk2qqp4n6j20pm0ixq88.jpg)

* master 正式版本分支，长期存在	

* develop 开发分支，长期存在

* hotfix 版本修复分支，是唯一能从master拉取的分支

* release 版本发布分支，发布新版本时使用

* feature 是从develop拉取的分支，用于日常开发
